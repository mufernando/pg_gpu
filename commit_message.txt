extend 'pairwise' missing data mode to all theta estimators

Add missing_data parameter to theta_h, theta_l, normalized_fay_wus_h,
zeng_e, zeng_dh, and fay_wus_h. All support 'pairwise'/'include'/
'exclude'/'ignore'. Per-site variable sample sizes used for theta_h
and theta_l; harmonic mean n for variance terms in H*, E.

Composite test statistics (H, H*, E, DH) use 'include' internally
when missing_data='pairwise' so all components are on the same
raw-sum scale. Refactored fay_wus_h to delegate to theta_h().
